#### GetInterfaceMetric

列出所有网络设备列表
#### 请求语法

```
GET /GetInterfaceMetric HTTP/1.1
```

#### 响应元素(Response Elements)
|名称|类型|描述|是否非空|
| ------------- |:-------------: |:-------------:| ------------- |
|Data|map  |返回网络接口信息              |非空|
|StatisticsTime|Integer|统计时间点，单位：秒 |非空|

#### InterfaceMetric
|名称|类型|描述|是否非空|
| ------------- |:-------------: |:-------------:| ------------- |
|ip |String|IP地址|非空|
|filter |String|用于过滤端口的名称 |可以为空 |

#### 请求示例

##### Request
```
GET /GetInterfaceMetric?ip=10.0.20.254&filter=GigabitEthernet   HTTP/1.1
```
